# 坦克大战 - Tank Battle

基于 HTML5 Canvas 的经典坦克大战网页游戏，纯前端实现，无需任何依赖。

## 游戏玩法

- 控制你的坦克消灭所有敌方坦克
- 保护地图底部的基地（⭐标记），基地被摧毁则游戏结束
- 每波敌人全灭后进入下一波，敌人数量和种类逐渐增加
- 收集道具获得临时能力提升

## 操作方式

| 按键 | 功能 |
|------|------|
| 方向键 / WASD | 移动坦克 |
| 空格 / J | 射击 |
| P | 暂停/继续 |

## 游戏特性

- **波次系统**：逐波递增难度，第2波起出现快速坦克（紫色），第3波起出现重甲坦克（橙色，需3次命中）
- **道具系统**：
  - ⚡ 加速 — 移动速度提升
  - 🔥 速射 — 射击间隔大幅缩短
- **地图**：随机生成砖块和墙壁布局，砖块可被子弹摧毁
- **复活机制**：3条命，被击中后有短暂无敌时间
- **视觉效果**：爆炸动画、闪烁效果、血条显示

## 运行方式

直接用浏览器打开 `index.html` 即可游玩。

```
# 或者使用任意 HTTP 服务器
python -m http.server 8000
# 然后访问 http://localhost:8000
```
